Welcome to libscapi
===================

Libscapi is an open-source c++ library for implementing secure two-party and multiparty computation protocols (SCAPI stands for the "Secure Computation API"). It provides a reliable, efficient, and highly flexible cryptographic infrastructure. SCAPI also has a java version, that can be found at `<https://scapi.readthedocs.io/en/latest/index.html>`_. SCAPI is free and is licensed under an adaptation of the MIT license, you can read more about the license :ref:`here <license>`.
